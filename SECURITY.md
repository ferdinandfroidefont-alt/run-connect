# Politique de divulgation responsable

Si vous pensez avoir trouvé une **vulnérabilité de sécurité** dans RunConnect :

1. **Ne pas** ouvrir une issue publique GitHub avec les détails d’exploitation.
2. Écrire à l’équipe à l’adresse de support indiquée dans l’application (paramètres → confidentialité / CGU) ou celle configurée via `VITE_PUBLIC_SUPPORT_EMAIL`.
3. Décrire le risque, les étapes de reproduction minimales, et si possible un correctif suggéré.

Nous nous efforçons de répondre dans un délai raisonnable. Merci de contribuer à la sécurité des utilisateurs.
